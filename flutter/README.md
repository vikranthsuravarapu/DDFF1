# 🌿 Grama Ruchulu Flutter App Structure

This directory contains the structure and key files for the Flutter mobile app.

## Folder Structure
- `lib/core/`: Theme, Constants, Router, API Client
- `lib/models/`: Product, Category, Order, User, etc.
- `lib/providers/`: Riverpod state management
- `lib/services/`: API, Auth, Razorpay, Notifications
- `lib/screens/`: All UI screens

## Key Files Provided:
- `pubspec.yaml`: Dependencies
- `lib/core/theme.dart`: App styling
- `lib/core/constants.dart`: API configuration
- `lib/services/api_service.dart`: Backend integration
